how to play:

1. have Steam running
2. install TF2 on Steam
3. installed [Source SDK Base 2013 Multiplayer](https://steamdb.info/app/243750/info/) on Steam (paste this into your browser: steam://install/243750) (or Tools section in the library, if that doesn't work)
4. [download the peak_tf mod](https://cdn.discordapp.com/attachments/1263345589081341973/1515893564113158225/peak_tf.zip?ex=6a32a3a5&is=6a315225&hm=daac85395222ee9651d5cf5fee15f91c4d4bb4f9e87946f251bb65f724895ffa&)
5. extract the zip literally anywhere
6. run (double click) ####click_me_to_start_game#### to start the game
7. from the server browser's internet tab join shounic's server

tip:
- this is a "fresh" installation of TF2 so everything is default. you may want to copy your cfg folder from TF2 into this
- [click this to see all weapon stats](https://docs.google.com/document/d/e/2PACX-1vSU1KNrPF8hUwCS-rCwWD1_RgCfRraHpiE8xRRLSRACcIxgurdIEpmKH0lTezQAnCE0_BOGo73rsrSj/pub)
- no item set buffs


**linux users:** run mod exe and sdk 2013 mp with proton
